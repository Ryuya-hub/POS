# テスト用バーコード

## 使用方法

このディレクトリに実際のバーコード画像（PNG/JPG）を配置してください。

または、以下のオンラインツールで生成したバーコードを印刷してカメラで読み取ってください：

1. https://barcode.tec-it.com/ja/EAN13
2. 商品コードを入力（例: 4901991001013）
3. 「バーコードを生成」をクリック
4. ダウンロードして印刷、またはスマートフォンで表示
